# Kylrix Connect & Peer-to-Peer Communication Roadmap 💬

This roadmap guides the milestones for encrypted real-time communications, audio/video rooms, and presence states.

## WebRTC Audio & Video Rooms
- [ ] Add support for screen sharing in real-time WebRTC huddles
- [ ] Implement adaptive bitrate streaming for crystal clear video in poor network conditions
- [ ] Build end-to-end encrypted audio-only voice channels with active speaker detection
- [ ] Design seamless external dial-in bridge options for remote guest access

## Encrypted Messaging & Presence
- [ ] Build offline messaging queue with client-side automatic replay buffer
- [ ] Optimize WebSocket real-time presence sync to eliminate connection lag
- [ ] Implement encrypted group-wide message threads with perfect forward secrecy
