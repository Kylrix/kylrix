# Kylrix Note & Knowledge Base Roadmap 📝

This roadmap outlines goals for markdown document authoring, smart tagging, and real-time document collaboration.

## Advanced Editor Features
- [ ] Add full WYSIWYG rich text toggle overlay to standard Markdown editor
- [ ] Build multi-cursor collaborative real-time editing using CRDTs
- [ ] Implement robust local-first autosave with deep revision history comparison
- [ ] Add code block execution playgrounds directly inside markdown notes

## Document Discovery & Links
- [ ] Design interactive visual graph representing inter-linked markdown notes
- [ ] Implement smart tag hierarchy groups with parent-child relationships
- [ ] Add client-side global fuzzy search indexing for instant document querying
