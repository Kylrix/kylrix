# Kylrix Security & Core Roadmap 🛡️

This roadmap outlines the milestones and standalone goals required to harden Kylrix's Sovereign Agentic Core, cryptographic primitives, and peer-to-peer security meshes.

## Cryptographic Operations & MEK Hardening
- [ ] Implement robust MEK backup with Shamir's Secret Sharing (SSS)
- [ ] Migrate local Pbkdf2 iteration settings dynamically to match client hardware capabilities
- [ ] Design hardware-backed secure storage (WebAuthn/Passkey based MEK recovery)
- [ ] Harden in-RAM Master Encryption Key (MEK) scrubbing on session lock

## WESP & Node Mesh Protocol
- [ ] Refactor Web Ecosystem Security Protocol (WESP) cross-origin frame messaging
- [ ] Implement strict token rotation on identity nodes
- [ ] Harden PIN piggybacking against persistent timing attacks
- [ ] Optimize mesh status heartbeats to minimize network overhead on background syncs
