# Kylrix Vault & Encryption Roadmap 🔒

This roadmap covers goals targeting the zero-knowledge credential vault, TOTP multi-factor storage, and offline synchronization mechanisms.

## Vault Secrets Hardening
- [ ] Implement client-side AES-GCM-256 field level encryption for all custom secret inputs
- [ ] Add native offline vault access with client-side cached encrypted snapshots
- [ ] Build automated password strength scoring and security leak alerts
- [ ] Add robust support for secure browser autofill extension integration

## TOTP & Authentication Upgrades
- [ ] Add direct QR-code scanning support to instantly spin up TOTP secrets
- [ ] Design seamless emergency read-only access delegation keys
- [ ] Optimize zero-knowledge Diffie-Hellman key exchanges for instant secret sharing
